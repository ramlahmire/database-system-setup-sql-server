# Database System Setup & SQL Project

This project was completed as part of my coursework and focuses on setting up a database environment and implementing a relational database using SQL Server.

## Project Overview
I configured a Windows Server environment, installed SQL Server, and created a database to manage student-related data.

## What I Did
- Installed and configured Windows Server 2019
- Promoted the server to a Domain Controller (Mississauga.local)
- Installed SQL Server 2019
- Created a database (StudentsDB114)
- Designed tables and relationships between entities
- Created and executed SQL queries with filtering (WHERE clause)
- Implemented indexing for performance optimization

## Database Design
- Created multiple related tables such as:
  - Students
  - Courses
  - Contacts
  - Accommodation Staff
  - Hall of Residence
- Designed relationships between tables to maintain data integrity

## Tools Used
- Windows Server 2019
- SQL Server 2019
- SQL (DDL, DML)

## What I Learned
- Database setup and configuration
- Designing relational databases
- Writing SQL queries and filtering data
- Improving performance using indexes
